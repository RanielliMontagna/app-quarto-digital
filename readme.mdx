## App - Quarto Digital

Projeto desenvolvido como finalização do curso de análise e desenvolvimento de sistemas na [UPF](https://www.upf.br/).

## Sobre o projeto

O projeto consiste em um sistema web para gestão de hotéis e similares, onde o usuário pode cadastrar quartos, reservas e clientes.

## Tecnologias utilizadas

- [React](https://pt-br.reactjs.org/) - Biblioteca JavaScript para criação de interfaces de usuário
- [TypeScript](https://www.typescriptlang.org/) - Superset JavaScript que adiciona tipagem estática
- [Styled Components](https://styled-components.com/) - Biblioteca para estilização de componentes
- [React Router](https://reactrouter.com/) - Biblioteca para criação de rotas
- [React Icons](https://react-icons.github.io/react-icons/) - Biblioteca de ícones
- [Cypress](https://www.cypress.io/) - Framework de testes end-to-end
- [Material UI](https://material-ui.com/) - Biblioteca de componentes React
- [React Hook Form](https://react-hook-form.com/) - Biblioteca para criação de formulários

Entre outras que podem ser encontradas no arquivo <code>package.json</code>

## Como executar o projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/en/) - Versão LTS ou superior instalada na máquina local

### Instalação

```bash
# clonar repositório
git clone

# entrar na pasta do projeto
cd app-quarto-digital

# instalar dependências
npm install ou yarn

# executar o projeto
npm start ou yarn start
```
